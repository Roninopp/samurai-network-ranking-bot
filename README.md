## 🛠️ Deployment

### Koyeb Deployment
1. Fork this repository
2. Go to [Koyeb](https://koyeb.com)
3. Create new app → GitHub → Select this repo
4. Add environment variable: `BOT_TOKEN=your_bot_token_here`
5. Deploy!

## 📝 Commands
- `/start` - Main menu
- `/wallet` - Check coins
- `/rankings` - Leaderboard
- `/shop` - Virtual store
- `/missions` - Daily tasks
